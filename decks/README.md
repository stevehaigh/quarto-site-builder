# Decks

Drop slide deck PDFs (preferred) or `.pptx` files in here.

Link to them from `projects.qmd` as `decks/your-file.pdf`.

## Why PDF?

- Renders inline in browsers — no download needed
- Far smaller than equivalent `.pptx` in git
- Stable across viewers

To export from PowerPoint: File → Export → PDF.

To export from Keynote: File → Export To → PDF.
